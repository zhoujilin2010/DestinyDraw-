## 这是什么？

你是否有过这样的幻想：走进彩票店，掏出一张精心计算过的号码，老板抬头看你一眼，眼神里闪过一丝敬畏——"这人是专业的"。

DestinyDraw 就是帮你实现这个幻想的。它看起来像一个军工级的数据分析平台，实际上也确实做了很多正经事：加密级随机数、历史数据对比、蒙特卡洛模拟、甚至接入了 AI 大模型来自动校准参数。它用了一切手段让你觉得**这次不一样**。

然后它内置了一个功能叫「**这辈子能中500万吗**」——真实概率运算的人生模拟器，它会告诉你：按概率，你得买几十年、花上百万，才能中一次一等奖。

**所以这是个什么项目？** 它是一个包装成"选号神器"的数学科普。你用它选号的时候觉得自己在掌控命运；你用完分析功能之后，就不再想买彩票了。两个功能我们都做到了极致。

## 功能地图

### 🤖 「我命由我不由天」系列（自动策略选号）

| 功能 | 你在干什么 | 实际在干什么 |
|------|-----------|-------------|
| **双色球智慧选号** | AI 策略分析推荐号码 | 随机生成 4 组号，扔一组，从剩下的里再随机 |
| **大乐透智慧选号** | 红蓝双维度智能杀号 | 同上，只是红球蓝球各扔一组 |
| **快乐8智慧选号** | 模拟真实摇奖过程出号 | 同上，但从 80 个球里摇 |

> 如果你觉得"杀号"这个词让你感觉自己在做数学分析——恭喜你，心理体验本身就是产品的一部分。

### ⚡ 「别废话快出号」系列（快速机选）

| 功能 | 说明 |
|------|------|
| **机选双色球** | 单式/复式/胆拖，支持部分自选 + 部分随机混搭 |
| **机选大乐透** | 同上，红球 1-35 / 蓝球 1-12 |
| **机选快乐8** | 选一到选十全玩法，单式/复式/胆拖 |

> 和彩票店老板的机选没有本质区别，但 UI 好看 100 倍，而且你会觉得自己"做了功课"。

### 📊 「面对现实」系列（历史数据分析）

| 功能 | 说明 |
|------|------|
| **错过100万了吗** | 输入你买过的号码，对比最近 100 期开奖，看看你曾经和奖金擦肩而过多少次 |
| **历史验证器** | 任意号码 vs 全量历史开奖，输出平均命中率和全空轮次——数据会告诉你答案 |
| **空号校验工具** | 蒙特卡洛 10 万注模拟，看看你的"策略"到底比随机强多少（答案通常是：不） |

> 这三个功能是我们真正的核心价值。选号是让你进来的，劝退是让你清醒的。

### 🎮 「认清现实」系列（人生模拟）

| 功能 | 说明 |
|------|------|
| **这辈子能中500万吗** | 基于真实概率运算，模拟一直买到中一等奖需要多少年、花多少钱、亏多少。会给你一段幽默但扎心的评语 |

> 这个功能值得单独拿出来说：它在 Web Worker 里跑，不卡页面。我们连让你绝望都考虑到了性能体验。

## 技术栈

- **纯前端**：一个 HTML + 一个 CSS + 一个 JS + 一个 Worker，零依赖零构建，浏览器直接打开就用
- **安全随机**：`crypto.getRandomValues` 而不是 `Math.random`——我们认真对待每一个注定不会中奖的号码
- **历史数据**：SSQ 3451 期 / DLT 2870 期 / K8 1232 期，通过 `update-data.py` 自动拉取
- **拖拽排序**：首页 10 个功能卡片支持长按拖拽重排，持久化到 localStorage
- **Web Worker**：人生模拟器在独立线程运行，一边模拟亏钱一边流畅滚动
- **响应式**：桌面 3 列 / 平板 2 列 / 手机 1 列
- **AI 校准**：接入 Kimi API 自动分析验证偏差并调整策略参数——虽然策略本质上还是随机，但我们连参数调整都很认真

## 常见问题

**Q: 你的自动选号真的能提高中奖概率吗？**  
A: 不能。没有任何东西能。如果我们的策略能提高概率，中国福利彩票就该改规则了。

**Q: 那为什么叫"AI 策略分析"？**  
A: 因为 AI 确实在分析——它分析你的历史验证偏差，然后调整杀号组数。只是这个调整对命中率的影响，几乎可以忽略不计。

**Q: 那这个项目到底有什么用？**  
A: 两个用途：(1) 让你在买彩票时心理感受更好；(2) 让你买完之后意识到不应该再买。先用 (1)，自然会到 (2)。

**Q: 你会用这个工具买彩票吗？**  
A: 作者本人偶尔花 2 块钱买一注，主要目的是在开奖时可以合法地做一会儿白日梦。工具是用来增加这 2 块钱的仪式感的，不是用来相信的。
