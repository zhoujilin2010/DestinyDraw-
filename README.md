# DestinyDraw 彩票助手

> 一款纯前端彩票选号工具，支持双色球、大乐透、快乐 8 三种彩票，提供机选、自动策略选号、历史验证等多种功能。

**在线地址**：[GitHub Pages](https://zhoujilin2010.github.io/DestinyDraw-/)  
**当前版本**：v1.12

---

## 功能总览

### 🎯 机选选号

| 功能 | 说明 |
|------|------|
| **机选双色球** | 支持单式、复式、胆拖三种玩法，可指定部分号码、剩余随机生成 |
| **机选大乐透** | 支持单式、复式、胆拖，红球 1-35 / 蓝球 1-12，灵活配置 |
| **机选快乐8** | 支持选一到选十全部玩法，可单式、复式或胆拖，支持自选 + 随机混搭 |

### 🤖 自动策略选号

| 功能 | 说明 |
|------|------|
| **双色球自动选号** | 生成 4 组参考号，以第 4 组为杀号组，从剩余球池中按选定玩法生成最终号码 |
| **大乐透自助选号** | 同双色球逻辑，红球 + 蓝球双维度杀号，从剩余球池出号 |
| **快乐8自动选号** | 生成 4 组参考号模拟摇奖，第 4 组作为杀号，按选几玩法出最终号码 |

### 📊 历史数据分析

| 功能 | 说明 |
|------|------|
| **错过100万了吗** | 输入历史持有号码（单式/复式/胆拖），对比最近 100 期真实开奖，统计曾错过的各等奖 |
| **历史验证器** | 输入任意一注号码，与双色球/大乐透/快乐8历史数据全量对比，输出平均命中率与全空轮次 |
| **空号校验工具** | 手动选号进行一次性历史比对，查看结果、AI 参数建议，支持回滚上一次策略 |

### 🎮 趣味模拟

| 功能 | 说明 |
|------|------|
| **这辈子能中500万吗** | 硬核彩票人生模拟器，基于真实概率运算，模拟一直买到中一等奖需要花多少年、多少钱 |

## 技术特性

- **纯前端**：零依赖、零构建，单 HTML + CSS + JS 文件，可直接本地打开运行
- **安全随机**：使用 `crypto.getRandomValues` 代替 `Math.random`，确保选号随机性更强
- **历史数据**：GitHub Actions 每日自动拉取最新开奖数据（`data/ssq.json`、`data/dlt.json`、`data/k8.json`），本地同步 localStorage 缓存（24h 有效）
- **拖拽排序**：首页 10 个功能卡片支持长按拖拽重排，布局自动持久化到 localStorage
- **Web Worker**：人生模拟器在独立 Worker 线程运算，不阻塞页面交互
- **响应式布局**：桌面 3 列 / 平板 2 列 / 手机 1 列自适应

## 目录结构

```
DestinyDraw-/
├── index.html          # 主页面（单页应用）
├── script.js           # 全部业务逻辑（约 4400 行）
├── style.css           # 全部样式（约 1900 行）
├── logo.png            # 品牌 Logo（透明通道 PNG）
├── CHANGELOG.md        # 中文更改记录
├── data/
│   ├── ssq.json        # 双色球历史开奖数据（GitHub Actions 自动更新）
│   ├── dlt.json        # 大乐透历史开奖数据
│   └── k8.json         # 快乐8历史开奖数据
└── scripts/
    ├── fetch-data.ps1  # PowerShell 数据拉取脚本
    ├── update-data.py  # Python 数据更新脚本
    └── setup-task.ps1  # Windows 计划任务配置脚本
```

## 本地运行

直接用浏览器打开 `index.html` 即可，无需任何服务器或构建步骤。

> 如需手动刷新彩票历史数据，运行：
> ```powershell
> .\scripts\fetch-data.ps1
> ```
