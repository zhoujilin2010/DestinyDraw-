## 一、Toast 通知组件（替换 alert()）

### 问题
审查报告中指出：项目中有 5 处 `alert()` 阻断式弹窗，用户体验差。

### 修改内容

**1. style.css (新增约 100 行)**
- 新增 `.toast-container` — 固定定位右上角的通知容器
- 新增 `.toast` — 通知卡片（带玻璃态效果、入场/退场动画）
- 新增 `.toast--success` / `--error` / `--info` / `--warning` 四种主题
- 新增 `@keyframes toastIn` / `toastOut` 动画
- 新增移动端响应式适配（`max-width: 480px`）

**2. script.js (第331-365行, 新增 showToast 函数)**
```javascript
function showToast(message, type) { ... }
```
- 支持 4 种类型：success / error / info / warning
- 每条 toast 3.2 秒自动消失
- 点击可立即关闭
- 自动创建容器 div，复用同一个容器

**3. 替换全部 5 处 alert() 调用：**

| 原位置 | 原代码 | 替换后 |
|--------|--------|--------|
| 导出无记录 | `alert('暂无记录可导出。')` | `showToast('暂无记录可导出。', 'warning')` |
| 导入格式错误 | `alert('文件格式不正确...')` | `showToast('文件格式不正确...', 'error')` |
| 导入成功 | `alert('✅ 导入成功...')` | `showToast('导入成功...', 'success')` |
| 解析失败 | `alert('文件解析失败...')` | `showToast('文件解析失败...', 'error')` |
| 策略更新 | `alert('✅ ...选号策略已更新...')` | `showToast('...选号策略已更新...', 'success')` |

**修改确认**: `grep -c "alert(" script.js` 返回 `0`，零残留。

## 二、Web Worker 代码提取

### 问题
审查报告中指出：~170 行 Worker 源码内联为 JavaScript 字符串（`LIFE_SIM_WORKER_SRC`），难以调试和维护。

### 修改内容

**1. 新建 `worker.js`** (5,853 字节，169 行)
- 包含人生模拟器蒙特卡洛运算完整逻辑
- 从 1-80 号球的无放回采样算法
- 单式/复式/胆拖三种投注模式的中奖判定
- 500,000,000 期上限的分块异步运算

**2. script.js 修改**
- 删除 `LIFE_SIM_WORKER_SRC` 模板字符串（原第1957-2103行，约147行）
- Worker 创建从 Blob URL 方式改为文件引用：
  ```javascript
  // 旧: const blob = new Blob([LIFE_SIM_WORKER_SRC], ...); new Worker(workerUrl);
  // 新: new Worker('./worker.js');
  ```
- 省去 `URL.createObjectURL` / `URL.revokeObjectURL` 开销

**收益**: 
- script.js 减少 ~150 行（净减少）
- Worker 代码可单独调试（浏览器 DevTools → Sources → worker.js）
- 可用编辑器语法高亮和格式化

## 四、innerHTML → DOM API 替换

### 问题
审查报告中指出：大量 `innerHTML` 模板字符串拼接降低可维护性。

### 修改内容

**1. 新增 DOM 助手函数 (script.js 第331-343行)**
```javascript
function domEl(tag, className, textOrChildren) { ... }
function setText(el, text) { el.textContent = text; return el; }
```

**2. 替换关键路径 innerHTML（6处）**

| 函数 | 原模式 | 替换方式 |
|------|--------|----------|
| `renderQuickPicker` | `header.innerHTML = \`<h4>...\`` | `domEl('h4', '', title)` |
| `renderReceiptResults` | `titleRow.innerHTML = \`<span>...\`` | `domEl('span', 'slip-title', '选号单')` |
| `renderReceiptResults` | `footer.innerHTML = \`<span>...\`` | `createElement` + `createTextNode` 拼接 |
| `renderQuickPage` | `switcher.innerHTML = \`<button>...\`` | `forEach` 循环创建 button |
| `renderQuickPage` | `actionBar.innerHTML = \`<button>...\`` | `createElement` + 属性设置 |
| `renderQuickPage` (2处) | `resultSection.innerHTML = '<h3>...'` | `domEl('h3', 'preview-title', '生成结果')` |
